# 🔥 HorizonChat

<div align="center">

A modern **real-time chat application** built with scalable backend architecture, instant messaging, and seamless user communication.

[🚀 Live Demo](https://horizon-chat-app.onrender.com/)

</div>

---

HorizonChat is a full-stack messaging platform that provides secure authentication, real-time conversations, media sharing, and social interactions with a smooth modern user experience.

---

# ✨ Features

## 🔐 Authentication & Security

- Secure user registration and login
- JWT authentication with access & refresh tokens
- HTTP-only cookie-based session handling
- Password hashing using bcrypt
- Email verification system
- Protected routes with authentication middleware
- API rate limiting for abuse prevention and enhanced security

---

## 💬 Real-Time Messaging

- Instant one-to-one messaging using Socket.IO
- Real-time message delivery without page refresh
- Persistent chat history
- Real-time conversation updates
- Typing indicators
- Online/Offline indicators
- Message read receipts
- Online user communication

---

## 📎 Media & Voice Support

- Image sharing in conversations
- File sharing support
- Voice message support
- Real-time media message delivery

---

## 👥 Friend System

- Search and discover users
- Send friend requests
- Accept/reject incoming requests
- Manage friendships
- Block users
- Relationship status tracking

---

## 🎨 User Experience

- Modern dark-themed interface
- Responsive chat layout
- Real-time sidebar updates
- User search and filtering
- Smooth chat interactions
- Clean and minimal messaging experience
